Building the baseline model with 16 hidden layers¶

In [8]:
hidden_layers=16

def baseline_model():
    # create model
    model = Sequential()
    model.add(Dense(num_pixels, input_dim=num_pixels, kernel_initializer='normal', activation='sigmoid'))
    model.add(Dense(hidden_layers, kernel_initializer='normal', activation='sigmoid'))
    model.add(Dense(num_classes, kernel_initializer='normal', activation='sigmoid'))
    #sgd = optimizers.SGD(lr=0.01, momentum=0.0, decay=0.0, nesterov=False)
    # Compile model
    model.compile(loss='mean_squared_error', optimizer='adam', metrics=['accuracy'])
    return model

In Summary :¶

- Increasing the number of neurons in the hidden layer and the number of iterations for the gradient descent has the same positive effect in the level of accuracy of the models as decreasing the number of samples/batch sizes for the mini batch gradient calclations in each layer.¶

- Adding another hidden layer with the same amount of neurons, makes no big difference to the model¶

- Bigger batch sizes for the mini batch gradient calclations has an adverse effect on the accuracy of the model.¶
